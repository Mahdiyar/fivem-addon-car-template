# Whats is this?

This is a **sample resource** to make it easier and faster for you to add your own **custom vehicles** to **FiveM** servers.

> **Tip:** This resource is compatible with **addon** and **replace**.

## How to use?

 - **Download** a vehicle From 5Mod.
 - **Extract** vehicle files.
 - Copy and put **fivem-addon-car-template** folder in your **server resources** folder.
 - Put the **.yft** and **.ytd** files in the **stream** folder.
 - Put the **.meta** files in the **data** folder.
 - start **fivem-addon-car-template** in **server.cfg**
 - Enjoy! :)

## 
Compatibility table with different types of vehicle files:

|      Type          |Support                         | Folders                         |
|----------------|-------------------------------|-----------------------------|
|Addon|`YES`|  Steam & DATA   |
|Replace|`YES`|    Only  Steam      |
|MLO|`NO`|   Unsupported   |
